# 各画像について判定する項目

```json
{
  "filename": "IMG_XXXX.jpg",
  "primary_subject": "室外機 / 室内機 / 分電盤 / コンセント / 配管 / 化粧カバー / 据置金具 / 屋根付き架台 / 工具 / 採寸 / 壁面(設備なし) / 配線 / 天井裏 / 床下 / その他",
  "location": "室内 / 室外 / 玄関 / 浴室 / 床下 / 天井裏 / 不明",
  "stage": "施工前 / 工程中 / 施工後 / 判定不能",
  "secondary_tags": ["屋根付き架台", "化粧カバー", "目隠しプレート", "化粧モール" などの該当する付属設備すべて],
  "composition": "全景 / 接写 / 中距離 / 手元",
  "quality": "good / blurry / dark / duplicate候補",
  "people_visible": true/false,
  "personal_info": true/false (表札、車のナンバー、お客様の顔等が読み取れるか),
  "description": "30〜80字で、写っているものを客観的に描写。形容詞は最小限",
  "best_use": "hero-before / hero-after / before / after / process / detail / 除外 のうち最適なもの",
  "suggested_caption": "ページに載せる時の短いキャプション案（10〜20字）"
}
```

# 判定のヒント

- **「施工前」**: 壁に穴がない / エアコン本体が映っていない / 元の壁面・元の場所の状態
- **「工程中」**: 道具・配管・配線が露出していて作業中の様子
- **「施工後」**: エアコン本体や室外機が設置完了している / 化粧カバーで配管が隠れている
- **「屋根付き架台」**: 室外機の上に金属の小さな屋根が付いている
- **「化粧カバー」 (室外化粧カバー / スリムダクト)**: 配管を覆う白い樹脂のカバー
- **「分電盤」**: ブレーカーが並んだ箱、家の中の電気の心臓部
- **「目隠しプレート」**: 壁に開けたケーブル貫通穴を隠す小さなプレート
- **連写グループ**: 似たような構図が連続している場合は best_of_group で1枚だけ true

# 出力形式

最終的に以下の形式で **JSONだけ** を返してください。前置きや結論文は不要。

```json
{
  "case": "{案件ID}",
  "case_name": "{案件の人間可読名}",
  "total_photos": {N},
  "photos": [
    { "filename": "IMG_XXXX.jpg", ... },
    ...
  ],
  "groups": {
    "hero_before_candidates": [...],
    "hero_after_candidates": [...],
    "before_after_pairs": [
      { "before": "IMG_XXXX.jpg", "after": "IMG_YYYY.jpg", "subject": "...", "note": "..." }
    ],
    "process_sequence": [...] (時系列順),
    "by_category": {
      "室内機": [...],
      "室外機": [...],
      "分電盤": [...],
      "配管・化粧カバー": [...],
      "工程・作業中": [...],
      "全景": [...],
      "除外候補": [...]
    }
  },
  "summary": "{120字以内の要約}"
}
```

# 注意

- 1枚ずつ Read tool で必ず実画像を見ること（推測で書かない）
- 並列でまとめて読むより、5〜10枚ずつ Read を並列発行 → 結果を整理、を繰り返すと効率的
- すべての判定根拠を持って、最後に1個の JSON にまとめる
- JSON 以外の説明文・前置きは不要
- 全件揃わない場合は欠落ファイルを明示
